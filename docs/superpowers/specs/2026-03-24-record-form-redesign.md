# 记一笔页面重构设计文档

## 1. 概述

### 1.1 背景

现有"记一笔"页面与设计文档存在较大差异，需要重构以对齐设计规范。

### 1.2 目标

- 完全对齐设计文档的布局和交互
- 采用页面内分类网格替代弹窗选择
- 补充缺失字段和功能
- 统一样式规范

### 1.3 方案

采用渐进式重构方案，在现有组件基础上逐步改造，保持代码可维护性。

---

## 2. 页面整体布局

### 2.1 结构

```
┌─────────────────────────────────┐
│ 顶部导航栏 (56px，固定)         │
├─────────────────────────────────┤
│ 分类标签栏 (44px)               │
├─────────────────────────────────┤
│ 分类图标网格 (内嵌)             │
├─────────────────────────────────┤
│ 金额输入区域                    │
├─────────────────────────────────┤
│ 详细信息区域                    │
├─────────────────────────────────┤
│ 标签显示区域                    │
├─────────────────────────────────┤
│ 备注输入区域                    │
└─────────────────────────────────┘
```

### 2.2 滚动行为

- 顶部导航栏固定
- 其他内容可滚动
- 无底部操作栏，保存通过顶部确认按钮完成

## 3. 组件设计

### 3.1 顶部导航栏

**布局：**
```
┌─────────────────────────────────┐
│ [×]      新增记录         [✓]  │
└─────────────────────────────────┘
```

**规格：**

| 属性 | 值 |
|------|-----|
| 高度 | 56px |
| 背景色 | #FFFFFF |
| 底部阴影 | 1px rgba(0,0,0,0.1) |
| 标题 | "新增记录" / "编辑记录" |
| 标题字号 | 17px, 600 |
| 标题颜色 | #000000 |
| 取消按钮 | × 图标，24px，#666666，左侧 |
| 确认按钮 | ✓ 图标，24px，#1890FF，右侧 |

**交互：**

| 操作 | 行为 |
|------|------|
| 点击取消 | 弹出确认对话框 → 确认后返回上一页 |
| 点击确认 | 验证必填项 → 提交数据 → 返回上一页 |

### 3.3 分类图标网格

**布局：**
```
┌─────────────────────────────────┐
│  ┌────┐  ┌────┐  ┌────┐        │
│  │图标│  │图标│  │图标│        │
│  │名称│  │名称│  │名称│        │
│  └────┘  └────┘  └────┘        │
│  ┌────┐  ┌────┐  ┌────┐        │
│  │图标│  │图标│  │图标│        │
│  │名称│  │名称│  │名称│        │
│  └────┘  └────┘  └────┘        │
└─────────────────────────────────┘
```

**规格：**

| 属性 | 值 |
|------|-----|
| 布局 | 3 列网格 |
| 容器内边距 | 16px |
| 图标项间距 | 16px |
| 图标容器 | 64x64px，#F5F5F5，圆角 8px |
| 图标大小 | 32px |
| 图标颜色 | #D4B16A（或分类自定义颜色） |
| 文字字号 | 12px |
| 文字颜色 | #333333 |

**选中状态：**

| 属性 | 值 |
|------|-----|
| 背景色 | #E6F7FF |
| 边框 | 2px #1890FF |

**图标方案：** 使用 Lucide 图标库，图标尽量贴合分类含义。

**交互流程：**
1. 用户点击分类标签栏切换类型
2. 分类图标网格内容随类型变化
3. 点击图标选中分类
4. 选中后图标显示选中状态

### 3.4 金额输入区域

**布局：**
```
┌─────────────────────────────────────────┐
│ [CNY]  [    金额输入框    ]  [+] [-]    │
└─────────────────────────────────────────┘
```

**规格：**

| 组件 | 属性 |
|------|------|
| **容器** | 背景 #FFFFFF，内边距 16px，间距 16px |
| **货币显示** | 文本 "CNY"，字号 14px，颜色 #666666，背景 #F5F5F5，圆角 4px，内边距 8px |
| **金额输入框** | 占位符 "0"，字号 32px，颜色 #000000，背景 #F5F5F5，圆角 4px，内边距 8px 16px |
| **加减按钮** | 图标 +/-，大小 24px，颜色 #666666，背景 #F5F5F5，圆角 4px |

**加减按钮功能：**

| 操作 | 行为 |
|------|------|
| 点击 + | 金额 +1 |
| 点击 - | 金额 -1（最小为 0） |
| 长按 + | 快速递增（每 100ms +1） |
| 长按 - | 快速递减（最小为 0） |

**货币：** 固定为 CNY，无选择功能。

### 3.5 详细信息区域

**布局（2列网格）：**
```
┌─────────────────────────────────────────┐
│ 名称        │ 账户                      │
│ [输入框]    │ [选择器]                  │
├─────────────────────────────────────────┤
│ 项目        │ 商家                      │
│ [输入框]    │ [输入框]                  │
├─────────────────────────────────────────┤
│ 次数        │ 日期                      │
│ [输入框]    │ [日期选择器]              │
├─────────────────────────────────────────┤
│ 时间        │                           │
│ [时间选择器]│                           │
└─────────────────────────────────────────┘
```

**字段说明：**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 名称 | 输入框 | 否 | 记录名称 |
| 账户 | 选择器 | 否 | 关联账户 |
| 项目 | 输入框 | 否 | 关联项目名称 |
| 商家 | 输入框 | 否 | 商家/商户名称 |
| 次数 | 数字输入 | 否 | 分期/重复次数 |
| 日期 | 日期选择器 | 是 | 记录日期，默认今天 |
| 时间 | 时间选择器 | 否 | 记录时间 |

**规格：**

| 属性 | 值 |
|------|-----|
| 容器背景 | #FFFFFF |
| 容器内边距 | 16px |
| 字段间距 | 16px |
| 标签字号 | 14px，颜色 #666666 |
| 输入框字号 | 14px，颜色 #333333 |
| 输入框背景 | #F5F5F5，圆角 4px，内边距 8px 16px |
